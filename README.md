# Ultima Vox Design Language

**Status:** Draft  
**Version:** 0.1.0 Foundation  
**Codename:** Foundation  

## What is UVDL?

Ultima Vox Design Language is an engineering language for designing digital products.

It is not only a UI kit, CSS framework, component library, or visual style guide.

UVDL exists to help teams make better product, UX, UI, content, accessibility, and frontend decisions with less guesswork and more validated knowledge.

## Motto

> Decisions before Components.

## Purpose

UVDL exists to reduce bad design decisions by turning intuition into structured, reusable, and testable knowledge.

## Current Milestone

### v0.1.0 — Foundation

The Foundation milestone defines project purpose, manifesto, north star, founding decisions, governance, versioning, RFC process, ADR process, knowledge model, terminology, and architecture.

No production CSS, components, or templates are introduced before the Foundation milestone is stable.

## First Reference Project

`mechaniki.ru` is the first planned reference implementation of UVDL.
